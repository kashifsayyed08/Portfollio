Project 1: Amazon CSS Clone

I developed an Amazon clone using HTML and CSS, comprising multiple pages such as the main index.html and product.html for detailed product views. The homepage mirrors Amazon's layout, featuring a familiar header with a logo, search bar, and navigation menu. It includes sections for showcasing featured products, product categories, and special deals. The product listings present each item in a card format, displaying images, titles, prices, and a "Buy Now" button. Users can filter and sort products based on categories and prices.

The product detail page, accessible through the "Buy Now" button, offers a comprehensive view of the selected product, including detailed information, images, prices, and customer reviews. Users can simulate the shopping experience by adding products to the cart, which is summarized in a shopping cart page, displaying details like product names, quantities, and total prices.
